A powerful WordPress plugin that provides database connectivity features with an intuitive tabbed settings interface.

== Description ==

Database Connector is a WordPress plugin designed to enhance your website's database connectivity capabilities. The plugin features a modern, tabbed settings interface and follows WordPress best practices for data management and user experience.

**Key Features:**
* Easy settings management with tabbed interface
* Custom page slugs configuration
* Data persistence across deactivation/uninstallation
* WordPress standards compliant
* Automatic setup and configuration

**Settings Location:**
Navigate to Settings > Database Connector in your WordPress admin to configure the plugin.

For detailed documentation, installation instructions, and developer information, please see the README.md file included with the plugin.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/db-conn` directory, or install the plugin through the WordPress plugins screen directly
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings > Database Connector screen to configure the plugin

== Frequently Asked Questions ==

= Will my settings be lost if I deactivate the plugin? =

No. This plugin follows a data preservation policy where all settings are permanently preserved, even after deactivation or uninstallation.

= Where can I configure the plugin settings? =

Go to Settings > Database Connector in your WordPress admin area.

= What are the default slug values? =

The default values are "login" for the login page slug and "dashboard" for the dashboard page slug.
